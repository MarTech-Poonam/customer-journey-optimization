# Customer Journey Optimization

## Client

Fictional online clothing retailer - Yeezee.com

## Problem

Customers purchase once and never return.

Repeat purchase rate is only 12%.

## Goal

Increase repeat purchases through customer journey automation.

## Technologies

- Salesforce CRM
- Marketing Cloud
- SQL
- GA4
- Email Automation

## Outcome

Designed automated lifecycle journey to improve customer retention.
